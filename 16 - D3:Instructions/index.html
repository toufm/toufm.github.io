Correlations between Healthcare and Poverty

The data visualization focuses on the health risks that particular demographics face in the United States. The scatterplot uses data from the U.S. Census Bureau and the Behavioral Risk Factor Surveillance System. The data set is based on 2014 ACS 1-year estimates. The interactive scatterplot shows the correlation between the percentage of poverty and access to health care according to each state. Data includes rates of income, obesity, poverty, etc. The D3.js library was used to create this scatterplot and d3-tooltips were added to interact with information in each state circle.
